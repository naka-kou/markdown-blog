# Markdown Blog

Next.jsとMarkdownファイルを使用して作成したシンプルなブログアプリです。

---

## 公開URL

https://markdown-blog-blond.vercel.app/

---

## 使用技術

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Vercel (デプロイ)

---

## 機能

- Markdownファイルから記事を読み込み
- 記事一覧ページ表示
- 記事詳細ページ表示
- サムネイル表示
- 本文100文字の抜粋表示
- 新しい記事順に並び替え
- レスポンシブなカードUI

---

## 開発環境

Node.js をインストール後、以下のコマンドで起動できます。

### 依存関係インストール

npm install

### 開発サーバー起動

npm run dev

ブラウザで以下を開きます。
http://localhost:3000

## デプロイ

Vercelを使用してデプロイしています。

GitHub → Vercel → 自動デプロイ
